Hearts & Paws

Timeline: Jan 2025 - Current

Role: Software Tester

Team: Team of 10

Capstone

Tech Stack: Next.js(React), TypeScript, Tailwind CSS, Prisma ORM, ESLint, Node.js, Jenkins, PostCSS, Middleware.

Testing Tools: Cypress, JavaScript, Jira, Git, Chrome Dev Tools.

Hearts & Paws is a feature-rich pet adoption platform built to simplify and personalize the adoption process for users and shelters alike. The application enables adopters to search pets based on breed, age, activity level, and compatibility, while also allowing shelters to manage listings, schedule appointments, and track adoption progress.
As a dedicated software tester, I led the testing efforts from end to end, focusing on ensuring seamless, bug-free user experiences. I developed detailed test cases and performed both manual and automated testing using Cypress, validating everything from UI components to deep user workflows like login, filters, and adoption forms.
I collaborated with the team in an Agile environment, using Jira for sprint planning and bug tracking, and contributed to version control using Git. I also actively tested RESTful API integrations and ran regression tests to ensure stability across builds.
My efforts helped uncover over 25+ issues early in development, improve cross-browser compatibility, and ultimately boost overall application stability by 30%. This project not only sharpened my technical testing skills but also strengthened my communication, collaboration, and attention to detail in a real-world development setting.

Meal Planner App

Timeline:Sep 2024 - Dec 2024

Role:Frontend Developer

Team:Team of 3

Course Project

Tech Stack: HTML, CSS, JavaScript, JQuery, Bootstrap.

The Meal Planner App is a full-featured web application designed to help users organize, plan, and track over 21 meals per week across breakfast, lunch, and dinner. It simplifies weekly meal organization while encouraging healthy eating habits and consistency.

As the frontend developer in a team of 3, I was responsible for designing a clean, responsive interface using HTML, CSS, Bootstrap, and JavaScript. I also implemented interactive components with jQuery and handled local storage to allow users to save their meal plans across sessions.

The app supports day-by-day visual planning, easy meal slot editing, and smooth interaction on both desktop and mobile devices. Reusable components and responsive layout ensured a seamless user experience.

This project refined my frontend development skills and gave me valuable experience in building user-centric features within a collaborative, real-world setting. 🍽️
